Trade-offs between Mesh and Point Cloud Rendering

Rendering quality: Mesh rendering usually has better quality than point clouds because mesh can express surface color, texture, light reflection, and gloss through its triangles.

Memory usage: Point clouds generally need less memory than meshes because they only store point positions, while meshes need to save vertex positions and face indices.
