Discuss some of the tradeoffs between rendering as a mesh vs a point cloud.
Rendering quality: mesh rendering usually has better quality than point cloud, this is because Mesh is better than point cloud because mesh has triangles, which can express surface color, texture, light reflection, and gloss
Memory usage: when storing the data, points cloud generally needs less memory than meshes. The main reason is we can only save the positions of points, while mesh data need to save the vertices positions and face indices.
